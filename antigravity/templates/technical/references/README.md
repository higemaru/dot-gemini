# 参考文献 — {{書籍タイトル}}

執筆に使った公式ドキュメント・記事・書籍などを記録しておく。

## 置くもの（例）
- 参照した URL とその内容メモ
- 書籍・記事の引用メモ
- 動作確認した環境・バージョン情報

## references.md（リスト管理）
参考文献が増えてきたら references.md を作ってリスト管理するとよい。

## フォーマット例
### 公式ドキュメント
- [Claude Code Docs](https://docs.claude.com) — スキル・コマンドの仕様

### 記事
- タイトル / URL / 参照日 / メモ

### 書籍
- 著者『書名』出版社

## 実行手順

1. CLAUDE.md を読んでジャンルを確認し、対応する DNA ファイルを読む
   - 小説: @~/.claude/rules/novel-dna.md
   - 技術書: @~/.claude/rules/technical-dna.md
2. 関連ファイルを読む
   - 小説: resources/characters.md と resources/plot.md
   - 技術書: resources/structure.md
3. 何を書くか確認する
   - 指示が曖昧な場合のみ聞く
   - 「どのシーンを書きますか？」
   - 具体的な指示があればそのまま進む
4. 初稿を書く
   - DNA の文体ルールに従う
   - 完璧を目指さない。まず書き切る
   - [要確認] タグ: 設定との矛盾・判断できなかった箇所に付ける
5. 書き終えたら末尾に付記する
   - 文字数: XX字
   - [要確認] がある場合はその一覧

## 注意
- 途中で止めない。最後まで書いてから見せる
- 「どうしますか？」は書き終えてから一度だけ

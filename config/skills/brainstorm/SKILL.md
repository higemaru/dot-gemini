## 実行手順

1. CLAUDE.md を読んでジャンル（小説 or 技術書）を確認する

2. 何についてブレストするか聞く
   - 例: 「どんなアイデアを出しますか？（プロット・シーン・キャラクター・テーマ など）」

3. 小説の場合は resources/characters.md と resources/plot.md を読む
   技術書の場合は resources/structure.md を読む

4. アイデアを **5案** 出す
   - 各案は 2〜4 行で。説明しすぎない
   - 案ごとに番号をつける
   - 良し悪しの評価はしない。発散が目的

5. 出し終えたら一言だけ聞く
   - 「気に入った案はありますか？ 深掘りしますか？」
   - 深掘りを求められたら、その案だけさらに展開する

## 注意
- アイデアを絞り込んだり、まとめたりしない
- 「これが最善です」と言わない
- あなたが選ぶ。Claudeは案を出すだけ
